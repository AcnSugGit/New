# Calculator Web Application

This is a simple web-based calculator built with HTML, CSS, and JavaScript.

## Features
- Basic arithmetic operations: addition, subtraction, multiplication, division
- Responsive and clean user interface
- Easy to use

## Usage
1. Open `index.html` in your web browser.
2. Use the calculator interface to perform calculations.

## How to Run Locally
- No installation required. Just clone/download the repository and open `index.html`.

## Project Structure
- `index.html` - Main HTML file
- `Calculator.js` - JavaScript logic for the calculator
